Análisis de Datos Biológicos con Machine Learning

Este proyecto se centra en el análisis de un conjunto de datos biológicos utilizando técnicas de machine learning supervisadas y no supervisadas.

Scripts de Análisis de Datos

Estructura del Repositorio

El repositorio incluye manejo de datos y montaje de DF a partir de varios ficheros fuentes (DATOS), scripts de análisis en Phytin y documentos de resultados en formato Markdown, html, y videos de gráficos obtenidos. Los nombres de ficheros y su organización se detalla en el README.

Configuración del Entorno

Se recomienda crear un entorno virtual de Python e instalar las dependencias necesarias

Se recomienda crear un entorno virtual de Python e instalar las dependencias necesarias listadas en requirements.txt para replicar el análisis de datos.

python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
