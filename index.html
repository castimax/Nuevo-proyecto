Análisis de Datos Biológicos con Machine Learning

Este proyecto se centra en el análisis de un conjunto de datos biológicos utilizando técnicas de machine learning supervisadas y no supervisadas.

Documentos de Resultados

  • Explicación ML No Supervisado
  • Explicación ML Supervisado
  • Explicación Métodos No Supervisados
  • Resultado Análisis de Clústeres
  • Análisis Modelos Supervisados y Resultados

Scripts de Análisis de Datos

  • comp_longitud.py: Comparación de Tamaños de Archivos
  • estudio2.py: Montaje en DataFrame y Análisis ML

Estructura del Repositorio

El repositorio incluye datos, scripts de análisis y documentos de resultados, organizados en carpetas específicas.

Configuración del Entorno

Se recomienda crear un entorno virtual de Python e instalar las dependencias necesarias.

python -m venv venv
source .venv/bin/activate
pip install -r requirements.txt
